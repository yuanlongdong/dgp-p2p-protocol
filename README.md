# DGP-P2P
去中心化 P2P 担保系统（Escrow + Guarantor + Dispute + Telegram Mini App + Web + Subgraph）。

## Local takeover
```bash
pnpm install
pnpm ops:takeover:check
```

## Ops commands
```bash
pnpm ops:healthcheck:arb
pnpm ops:healthcheck:op
pnpm demo:prepare:arb
pnpm demo:prepare:op
pnpm demo:e2e:arb
pnpm demo:e2e:op
```
